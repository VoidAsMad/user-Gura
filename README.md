## Role
1. `/addrole [role] [user]`으로 유저에게 역할을 부여할 수 있습니다.
2. `/remove_role [role] [user]`으로 유저에게 역할을 제거 할 수 있습니다.

## User
1. `/kick [user] (reason)`으로 유저를 추방할 수 있습니다(재입장 가능)
2. `/ban [user] (reason)`으로 유저를 차단할 수 있습니다(재입장 불가능)
